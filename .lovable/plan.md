

# 📌 Repositório de Links

Um app pessoal para salvar, organizar e acessar rapidamente seus links favoritos da internet — sem necessidade de login, com dados salvos no navegador.

## Páginas e Layout

- **Página principal** com lista de todos os links salvos
- **Barra lateral** com categorias e tags para navegação rápida
- **Formulário** para adicionar/editar links

## Funcionalidades

### Salvar Links
- Adicionar link com URL, título, descrição e categoria
- **Preview automático**: ao colar uma URL, o app tenta mostrar o título e favicon do site
- Editar e excluir links existentes

### Organização
- **Categorias** (ex: Trabalho, Estudos, Lazer) — criar, renomear e excluir
- **Tags** — adicionar múltiplas tags a cada link
- Filtrar links por categoria, tag ou ambos

### Favoritos
- Marcar/desmarcar links como favoritos (ícone de estrela)
- Seção de favoritos no topo ou na barra lateral para acesso rápido

### Busca
- Campo de busca para filtrar links por título, URL ou descrição

## Armazenamento
- Todos os dados salvos localmente no navegador (localStorage)
- Sem necessidade de conta ou backend

## Design
- Interface limpa e moderna com cards para cada link
- Visualização em grade ou lista
- Responsivo para funcionar bem no celular e desktop
